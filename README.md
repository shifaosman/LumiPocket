# LumiPocket

A handcrafted, frontend-only startup-style demo built with Next.js App Router, TypeScript, Tailwind CSS, Framer Motion, Zustand, and Radix UI.

## Getting Started

Install dependencies and start development:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build and lint

```bash
npm run lint
npm run build
```

## Routes

- `/` Home
- `/studio` Companion Studio
- `/themes` Theme Lab
- `/components` UI Showcase
- `/about` About

## Highlights

- Persistent floating companion across routes
- Click-to-explain UI guidance bubbles with fallback explanations
- Token-driven theming with curated presets and custom editors
- Responsive polished surfaces, motion, and accessibility-first controls
